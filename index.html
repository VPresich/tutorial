Для того, щоб запушити проект у репозиторій GitHub, потрібно виконати наступні кроки:

  • Відкрити термінал у папці з проєктом.
  • Додати файли до репозиторію: виконай команду git add (. для додавання всіх файлів з поточного каталогу).
  • Зробити коміт: виконай команду git commit -m "коментар", де "коментар" - це опис змін у коміті.
  • Створити віддалений репозиторій на GitHub та скопіювати його URL.
  • Додати посилання на віддалений репозиторій: відкрий термінал і введи команду git remote add origin , де - це URL віддаленого репозиторію, який ти створив на GitHub.
  • Запушити зміни: виконай команду git push -u origin main. Це додасть твої зміни до віддаленого репозиторію. Будь ласка, переконайся, що у тебе встановлено Git та ви користуєтесь правильною командою, щоб уникнути будь-яких помилок.
